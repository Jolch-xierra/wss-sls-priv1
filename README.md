# My Purpose, My Future – Primary Edition (Hero Journey)

This is a self-contained web version of the **“My Purpose, My Future – Primary Edition”** module designed for Primary 5–6 students. It guides pupils through a short hero journey to discover their interests, strengths, values, a problem they care about, and a simple purpose statement plus one small action.

## What this page does

- Presents a 5-step **hero journey**:
  1. Start – What success means to me  
  2. Discover Me – My interests and styles  
  3. Superpowers – My strengths and values  
  4. Big World – A problem I care about  
  5. My Mission – Purpose statement and small step  

- Includes a **Mission Control** screen where pupils build a sentence:
  > I want to use my [style] and my strength of [strength] to help with [problem] so that [impact].

- Generates a printable **“My Purpose Card – P5/6 Explorer”** with:
  - Name and class  
  - Styles and superpowers  
  - Problem they care about  
  - Purpose statement  
  - One small step for this term  
  - A drawing box: “Me living out my purpose”

Everything runs in the browser with a single `index.html` file (no React, no build step).

## How to run locally

1. Download or clone this repository.
2. Open `index.html` in any modern browser (Chrome, Edge, Firefox, Safari).
3. Interact with the screens using mouse or touch.
4. Click **“Print / Save as PDF”** on the Purpose Card screen to print or save the card.

No additional installation is required.

## How to deploy with GitHub Pages

1. Commit `index.html` (and this README) to your GitHub repository’s `main` branch.
2. In GitHub, go to  
   **Settings → Pages**.
3. Under **Build and deployment**, set:
   - Source: `Deploy from a branch`
   - Branch: `main`  
   - Folder: `/ (root)`
4. Click **Save**.  
5. After a short while, GitHub Pages will provide a URL where the module is live.

You can share this URL with teachers and students, or embed it as an external resource in SLS.

## Suggested classroom use

- Use it as a **single lesson (30–40 minutes)** in a computer lab or 1:1 device setting.
- Encourage pupils to:
  - Talk through each step with a partner.
  - Print or save their Purpose Card at the end.
- Teachers can collect printed cards as part of pupils’ portfolio evidence for purpose/values education.

## Accessibility and devices

- Designed for laptops and larger tablets.
- Uses large buttons and clear text for upper primary.
- Works offline once `index.html` is loaded (no external build tools).
